# ADDER



The adders are the simple  combinational circuits which are used to add binary data. the half adder performs addition on two input bits providing the sum and carry output, where as in full adder it takes a additional carry input along with the two input binary bits. The carry input in the Full adder comes from the previous addition operation. 

The source file folder contains the Verilog codes of half adder and full adder with structural modeling or gate level modeling. the testbench for both half adder and full adder are in the testbench folder.
